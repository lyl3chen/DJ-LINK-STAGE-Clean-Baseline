## 05:06 Ableton Link “connection refused” 频繁弹窗解释
- 用户反馈 Ableton Link 经常弹出 `unable to connect to carabiner port 17000: Connection refused`。
- 已解释本质：
  - Java 侧连接 127.0.0.1:17000 时，Carabiner 当下未监听（本地 TCP 时序空窗）
  - 常见于启动竞态、配置保存导致的 stop/start 窗口、子进程瞬时重启
- 结论：更多是“连接时序瞬态”而非协议逻辑错误。
- 已提出后续稳态化方案（待用户确认）：
  - 对 `connection refused` 先显示“重连中/启动中”
  - 仅在连续失败超过阈值（如 3~5 秒）才升格为红色错误
